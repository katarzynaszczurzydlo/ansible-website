Simple Website

Created using CloudFormation and Ansible

CloudFormation

Infrastructure was set up using simple CloudFormation template.

The template creates two Ec2 instances and a LoadBalancer.

Ansible

Ansible was used to install packages and upload the webpage

To make it work you need to provide path to your private_key_file in ansible.cfg and change the webservers PublicIP addresses in hosts-dev file to your Ec2 instances created via CloudFormation template.
